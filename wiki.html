Getting Started

Welcome to Brezel Minecraft! This guide contains essential information to begin your journey on our server.

Server IP
play.brezel.com

Quick Start Guide

  1. Launch Minecraft Java Edition
  2. Click Multiplayer > Add Server
  3. Enter the server address above
  4. Join the server and begin your adventure!

Survival Mode

Our survival mode combines classic Minecraft gameplay with custom plugins and a dynamic economy system.

Features

  • Advanced economy system
  • Custom crafting recipes
  • Unique items
  • PvP and PvE zones

Skyblock

Create your island empire, develop it, and trade with other players!

Island Commands

  • /island create - Create new island
  • /island home - Teleport to island
  • /island invite [player] - Invite player
  • /island settings - Island settings

Factions

Build your clan and engage in epic territorial warfare!

Basic Commands

  • /f create [name] - Create new faction
  • /f invite [player] - Invite player
  • /f claim - Claim territory
  • /f map - Show faction map

Economy System

Our server features a sophisticated economy system.

Currency Commands

  • /money - View your balance
  • /pay [player] [amount] - Transfer money
  • /shop - Open market menu
  • /sell - Sell items

General Commands

Essential commands available on our server:

  • /spawn - Teleport to spawn
  • /tpa [player] - Send teleport request
  • /sethome - Set home location
  • /home - Teleport to home
  • /msg [player] [message] - Send private message
